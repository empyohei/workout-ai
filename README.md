# 💪 Workout AI - HEVY Style Workout Tracker

Auto-analyze gym workouts with AI. Built with React, Gemini Vision API, and Cloudflare.

## ✨ Features

- **Home**: View workout history and stats
- **Workout**: Auto-recognize exercises from photos using Gemini Vision AI
- **Profile**: Track personal records and workout statistics

## 🚀 Quick Start

### 1. Get Gemini API Key

Visit: https://aistudio.google.com/app/apikey

### 2. Open the App

Visit your Cloudflare Pages URL

### 3. Enter API Key

Paste your API key and click "Get Started"

### 4. Start Logging Workouts

Take photos of your workout menus and let AI analyze them!

## 💻 Tech Stack

- **React 18** + **Vite**
- **Tailwind CSS** for styling
- **Google Gemini Vision API** for AI analysis
- **Cloudflare Pages** for hosting
- **localStorage** for data storage

## 🎨 Design

HEVY-inspired dark theme with:
- Slate color palette
- Blue accents
- Bottom tab navigation
- Mobile-first responsive design

## 📊 Data

All data stored locally in browser localStorage:
- Workouts
- User profile
- Gemini API key

No backend required. No data sent to servers.

## 🔐 Privacy

- Your workouts stay on your device
- Gemini API key stored locally in browser
- HTTPS encrypted communication
- No cloud sync

## 📱 Mobile

- iPhone: Open in Safari → Share → Add to Home Screen
- Android: Open in Chrome → Menu → Install app

## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| "API key not valid" | Get new key from aistudio.google.com |
| Workouts not saving | Check browser localStorage is enabled |
| Analysis failed | Take clearer photo, check API key |
